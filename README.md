# ABM Replication — Social Influence Model

## Purpose
Partial replication of [Dynamics of Public Opinion: Diverse Media and Audiences' Choices (Chen & Lan, 2021)](https://www.jasss.org/24/2/8.html) for POL42340. Aim was to learn and practice object oriented programming.

## Requirements
- random
- math
- json
- datetime

## Structure

| File | Description |
| --- | ---|
| `agent.py` | shared base behaviour of medium and person |
| `medium py` |
| `person.py` |
| `log.py` | Append timestamped messages to file |
| `configuration.py` and `config.json` | simulation parameters |
| `simulation.py` | set up network of agents and runs simulation |
| `main.py` | runs a simulation instance |
